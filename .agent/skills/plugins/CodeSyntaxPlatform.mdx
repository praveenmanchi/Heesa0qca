```
type CodeSyntaxPlatform =    "WEB" |    "ANDROID" |    "iOS"
```

[`codeSyntax`](/docs/plugins/api/Variable/#codesyntax) is a variable field used for defining custom token names for up to three unique code platforms. There are no formatting or syntax restrictions enforced on the code syntax entries.
